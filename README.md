# UNI-FIEE - Curso de Seguridad de Software 2026-I

Repositorio con laboratorios prácticos orientados al aprendizaje de conceptos fundamentales de Seguridad de Software, vulnerabilidades web y seguridad en la cadena de suministro de software.


## Objetivo

Desarrollar competencias prácticas en identificación, explotación controlada y remediación de vulnerabilidades de seguridad presentes en aplicaciones modernas, alineadas con OWASP Top 10 y OWASP API Security Top 10.

## Uso

Cada laboratorio contiene ejemplos, código fuente y ejercicios orientados al análisis y mitigación de vulnerabilidades específicas.

## Contenido

### Laboratorio 04 - Inyección
- Lab04-Inyeccion-0
- Lab04-Inyeccion-1-Java Web
- Lab04-Inyeccion-2-Spring Boot

Temas:
- Inyección de comandos y consultas.
- Validación de entradas.
- Prácticas seguras de desarrollo.

### Laboratorio 07 - Vulnerabilidades Web
- Lab07-1-CSRF-Hacker-Cliente
- Lab07-1-XSS-Reflejado

Temas:
- Cross-Site Request Forgery (CSRF).
- Cross-Site Scripting (XSS).
- Protección de aplicaciones web.

### Laboratorio 09 - Seguridad de la Cadena de Suministro
- Lab09-1-SBOM-SCA

Temas:
- Software Bill of Materials (SBOM).
- Software Composition Analysis (SCA).
- Gestión de dependencias y vulnerabilidades.

### Laboratorio 10 - Seguridad de APIs
- Lab10-1-BOLA-BPLA

Temas:
- Broken Object Level Authorization (BOLA).
- Broken Property Level Authorization (BPLA).
- Controles de autorización en APIs.
